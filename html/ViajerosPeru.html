Viajeros Perú
Viajeros Perú es una empresa peruana dedicada a la venta de paquetes turísticos en el Perú. Ofrecemos una amplia variedad de destinos y experiencias para que nuestros clientes disfruten de unas vacaciones inolvidables.

Este proyecto fue desarrollado utilizando las siguientes tecnologías:

  • Java
  • CSS
  • HTML
  • JavaScript

El objetivo principal del proyecto es proporcionar una plataforma fácil de usar para que los usuarios puedan explorar y reservar paquetes turísticos en línea.
